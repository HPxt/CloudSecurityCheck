# Trilha Expert (prática)

Esta trilha é para operação real de segurança em cloud, com foco em **resposta a incidentes** e **decisão rápida sob pressão**.

## Objetivo

Sair de “conhecer controles” para “executar resposta consistente” em cenários críticos.

## Conteúdo

- `playbooks/credencial_comprometida.md`
- `playbooks/bucket_publico.md`
- `playbooks/escalacao_privilegios_iam.md`

## Como usar esta trilha

1. Escolha um playbook por risco prioritário do seu ambiente.
2. Rode tabletop exercise com time técnico e gestor.
3. Ajuste passos para sua arquitetura (contas, projetos, SIEM, on-call).
4. Versione melhorias no próprio playbook após cada simulação/incidente.

## Resultado esperado

- Menor tempo de detecção e contenção.
- Decisões mais previsíveis durante crise.
- Evidência de maturidade operacional para auditoria.
