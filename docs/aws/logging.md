# AWS Logging - por que logs são essenciais

Sem logs confiáveis, você não sabe o que aconteceu em um incidente.

## Baseline recomendado

1. CloudTrail habilitado em múltiplas regiões.
2. Validação de integridade de logs.
3. Retenção e armazenamento protegidos.

## Benefício para o time

- Investigação mais rápida.
- Evidência para auditoria.
- Menos suposição, mais fato.
