# Trilha Intermediária

Arquivos desta pasta:

- `aws.md`: evolução de IAM, logging, networking e governança Terraform para AWS.
- `gcp.md`: evolução de IAM, logging e rede para GCP.

## Quando usar esta trilha

Use este material quando:
- você já implementou o baseline básico;
- seu time precisa reduzir risco operacional e não só cumprir checklist;
- você quer preparar o terreno para práticas avançadas.
