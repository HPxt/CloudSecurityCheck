# CloudSecurityCheck

Checklist básico para Cloud Security Engineers.

## Objetivo

Este projeto reúne uma lista prática de verificações para fortalecer a segurança em ambientes de nuvem.

## Tópicos sugeridos

- Gestão de identidade e acesso (IAM)
- Proteção de dados (em trânsito e em repouso)
- Monitoramento e resposta a incidentes
- Hardening de serviços e workloads
- Conformidade e auditoria contínua
